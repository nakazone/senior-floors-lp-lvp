# LVP textures for 3D showroom

Optional: place texture images here for the 3D viewer (e.g. `oak-light.jpg`, `oak-natural.jpg`, `walnut-dark.jpg`, `gray-modern.jpg`).

The app currently uses images from `/assets/` (see `src/lib/lvpTextures.ts`). To use images from this folder, update the `texture` and `thumbnail` paths in `lvpTextures.ts` to `/lvp/your-file.jpg`.

Use compressed images (e.g. JPEG) for best performance.

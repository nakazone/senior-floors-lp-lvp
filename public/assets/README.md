# Assets

Add **SeniorFloors.png** here so the logo loads from your site instead of senior-floors.com.

The Header and Footer use `https://senior-floors.com/assets/SeniorFloors.png` by default. To use a local file, update the `src` in `src/components/Header.tsx` and `src/components/Footer.tsx` to `/assets/SeniorFloors.png`.
